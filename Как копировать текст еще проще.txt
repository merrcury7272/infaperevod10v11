Установка и настройка

1. Установи необходимые пакеты

Открой терминал и выполни:

sudo apt update
sudo apt install tesseract-ocr tesseract-ocr-rus xclip gnome-screenshot

Примечание: Если используешь Wayland (например, в Ubuntu 22.04+), вместо xclip установи wl-clipboard:

sudo apt install wl-clipboard

2. Создай скрипт ocr_clip.sh

Сохрани следующий код в файл ocr_clip.sh:

НАЧАЛО КОПИРОВАНИЯ СНИЗУ

#!/bin/bash

 
TMPDIR=$(mktemp -d)

# Захватываем выбранную область экрана и сохраняем в файл
gnome-screenshot -a -f "$TMPDIR/screenshot.png"

# Распознаём текст с помощью Tesseract (русский и английский языки)
tesseract "$TMPDIR/screenshot.png" "$TMPDIR/output" -l rus+eng

# Копируем результат в буфер обмена
if command -v wl-copy &> /dev/null; then
    cat "$TMPDIR/output.txt" | wl-copy
elif command -v xclip &> /dev/null; then
    cat "$TMPDIR/output.txt" | xclip -selection clipboard
else
    echo "Ошибка: не найдено ни 'wl-copy', ни 'xclip'. Установи один из них."
    exit 1
fi

# Удаляем временные файлы
rm -r "$TMPDIR"

echo " Текст скопирован в буфер обмена!"

КОНЕЦ КОПИРОВАНИЯ 


3. Сделай скрипт исполняемым

Выполни в терминале:

chmod +x ocr_clip.sh


4. (Опционально) Перемести скрипт в системную директорию

Чтобы запускать скрипт из любого места, перемести его в /usr/local/bin:

sudo mv ocr_clip.sh /usr/local/bin/ocr_clip

Теперь ты можешь запускать его командой:

ocr_clip

4. (Опционально) Назначь горячую клавишу

Чтобы запускать скрипт по нажатию клавиш:
 1. Открой «Настройки» → «Клавиатура» → «Горячие клавиши».
 2. Нажми «+» для добавления новой комбинации.
 3. Назови, например, OCR Clip.
 4. В поле команды введи:

/usr/local/bin/ocr_clip


5. Назначь удобную комбинацию клавиш, например, Ctrl+Shift+O.

✅ Как использовать
 1. Нажми назначенную горячую клавишу или запусти команду ocr_clip в терминале.
 2. Выдели область экрана, содержащую нужный текст.
 3. Готово! Текст автоматически скопирован в буфер обмена. Вставь его с помощью Ctrl+V.
