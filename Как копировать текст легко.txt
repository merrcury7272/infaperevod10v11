Как распознать текст с экрана в Linux (через скриншот + OCR)

Инструкция позволяет скопировать любой текст, который нельзя выделить мышкой или скопировать обычным способом.

1. Сделай скриншот нужной области

Ты можешь использовать любую утилиту:

Вариант A (по умолчанию в Ubuntu):

gnome-screenshot -a

Выдели нужную область экрана. Скриншот сохранится в папке ~/Изображения или ~/Pictures.

Вариант B (если установлен flameshot):

flameshot gui

Выдели область, нажми Enter — и сохрани файл.

2. Установи программу распознавания текста (Tesseract OCR)

В Ubuntu или Debian:

sudo apt update
sudo apt install tesseract-ocr tesseract-ocr-rus

 • tesseract-ocr — сама программа
 • tesseract-ocr-rus — поддержка русского языка
(если нужен только английский — tesseract-ocr достаточно)

Проверь, что всё работает:

tesseract --version


3. Применить распознавание к скриншоту

Перейди в папку, где лежит скриншот, или укажи полный путь.

tesseract screenshot.png output -l rus+eng

Объяснение:
 • screenshot.png — имя твоего скриншота (замени на своё!)
 • output — имя выходного файла (создастся output.txt)
 • -l rus+eng — языки: русский + английский (можно eng или rus отдельно)

4. Посмотреть распознанный текст

cat output.txt

Готово!

Теперь ты можешь скопировать текст откуда угодно — даже если это терминал, PDF, системное окно, где нельзя выделить текст мышкой.


Пример полного использования:

gnome-screenshot -a             # делаю скриншот
tesseract ~/Изображения/screenshot.png text -l rus+eng
cat text.txt                    # читаю результат
